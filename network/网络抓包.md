# 一. tcpdump命令

[tcpdump](http://linux.51yip.com/search/tcpdump)是一款强大的网络抓包工具，它支持针对网络层、协议、主机、网络或端口的过滤，并支持and、or、not等逻辑语句。

```shell
tcpdump [ -DenNqvX ] [ -c count ] [ -F file ] [ -i interface ] [ -r file ]
        [ -s snaplen ] [ -w file ] [ expression ]

```
